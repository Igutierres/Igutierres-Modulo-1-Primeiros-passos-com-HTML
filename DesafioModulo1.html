História dos Computadores

Tudo começou em 1944 com o Harvard Mark I, o primeiro computador, posteriormente vieram os outros, conhecidos como Colossus (fabricado em1946) e por ultimo mas nao menos importante o ENIAC (também fabricado em 1946), marcando assim o fim da Primeira Geração de computadores.

A Segunda Geração de computadores (1959-1964) foi caracterizada pela criação dos transítores, que vieram a substituir as válvulas eletrônicas. Fazendo com que os computadores ocupassem menos espaços, serem mais eficientes e além disso, mais baratos.

A Terceira Geração de computadores deu-se ao início em 1965 e sua principal evolução foi devido à subistituição de transitores por circuitos integrados, teve também os primeiros Chips. Com isso os computadores ficaram ainda mais compactos reduzindo seu tamanho e seu peso, que ainda ficava acima de 20kg.

A Quarta Geração de computadores começou em 1971 e permeia até os dias atuais. Quando a Intel lançou o primeiro microprocessador, o Intel 4004, muito melhor que os circuitos SSI e MSI de até então.

O que são Clients

Em poucas palavras podemos dizer que Client é como um terminal burro que aproveita os recursos do computador que está associado. É um computador com configuração de hardware e software enxuta conectado em um servidor que hospeda aplicações e arquivos utilizados por ele, todo o processamento é feito pelo servidor, cabendo ao thin client apenas a exibição na tela do que é executado.

O que são Servers

Existem varios tipos de Servidores e eles geralmente são utilizados para:

  • Arquivos;
  • Segurança(Firewall);
  • Streaming;
  • E-mail;
  • Web.

E podem ser tanto um computador simples num canto da sala quanto uma maquina enorme, que geralmente é chamada de DataCenters, ou seja, um conjunto de servidores que serve para servir os usuários que estão acessando eles.

Linguagens de Programação

Linguagem de Programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas(Software).

Segundo o site RedMonk as linguagens de programação mais utilizadas atualmente são

  1. JavaScript
  2. Python
  3. Java
  4. PHP
  5. C#

Mas isso não significa que uma linguagem é melhor que a outra, depende de qual você pretende utilizar no seu projeto. Linguagem de Programação é uma ferramenta para resolução de problemas.

Lista Ordenada e não Ordenada

A principal diferença entre a Lista Ordenada e a Lista não-Ordenada está na maneira com que os itens de cada lista são apresentados.

Exemplo de Lista Ordenada:

  1. item 1
  2. item 2
  3. item 3

Exemplo de Lista não Ordenada:

  • item 1
  • item 2
  • item 3
